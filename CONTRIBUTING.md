# 贡献指南

感谢你对AI Podcast项目的关注！

## 如何贡献

### 报告Bug

在[Issues](https://github.com/yourusername/ai-poadcast/issues)中提交，包含：
- 问题描述
- 复现步骤
- 预期行为
- 实际行为
- 环境信息（Python版本、操作系统）

### 提交功能请求

在Issues中描述：
- 功能用途
- 使用场景
- 预期效果

### 提交代码

1. Fork项目
2. 创建特性分支：`git checkout -b feature/amazing-feature`
3. 提交更改：`git commit -m 'Add amazing feature'`
4. 推送分支：`git push origin feature/amazing-feature`
5. 提交Pull Request

## 代码规范

- 遵循PEP 8
- 添加类型注解
- 编写文档字符串
- 保持函数简洁（<50行）
- 添加单元测试

## 测试

```bash
# 运行测试
pytest tests/

# 运行示例
python examples/dependency_injection_demo.py
```
